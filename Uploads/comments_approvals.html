Help & Information

Admin Approval

Click the on the left to open the Admin Approval panel. This panel is only visible to admins and displays the approval history for each account, along with the upcoming approval schedule. During this time, admins will use this panel to review and approve the prior month's data before it becomes visible to the client. If an admin is associated with multiple accounts, the approval history will be grouped by account and listed in alphabetical order.

  • The monthly financial data is available for admin review beginning on the 7th of the month unless it is shifted to account for holidays or weekends.
  • If admins decide to hold the data, a message explaining why is required. Held data can be approved at any time. The admin will have the opportunity to file a support ticket with E15 for help with data issues if a hold has been placed.
  • Approved data will populate the next day in the dashboard for all users.
Summary Message

Once data has been approved or held, the admin will have the opportunity to write a summary message for the client regarding the monthly data. This message will appear on top of the dashboard upon opening the page. Messages associated with prior months will be visible in the side panel.

  • Admins are able to include one message per account per month.
  • Click on the left to review messages. Admins can also add, edit, or delete the latest month's messages. Access the Add Client Message interface by clicking at the top of the panel. The message will be applied to the individual account selected in the dashboard. Messages cannot be applied across multiple accounts.
  • Click to edit a message or to delete.
Admins

Admins are selected by internal stakeholders during the onboarding process and can be updated at any time by contacting E15. Accounts can have more than one admin to help with coverage for situations such as out of office days.
